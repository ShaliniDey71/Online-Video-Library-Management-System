# Online Video Library Management System

The Online Video Library Management System is a web-based platform designed to manage and facilitate the borrowing and returning of videos. It provides access to different types of users including members, librarians, and managers.

## Features

- **Member Users**: Members can register and log in to the system. Once logged in, they can browse the video library, borrow videos, and return them.
- **Librarians**: Librarians have additional functionalities such as managing the video inventory, adding new videos, and updating video information.
- **Managers**: Managers have the highest level of access, allowing them to oversee the entire system, generate reports, and manage member accounts.

This system aims to streamline the process of video library management, making it efficient and user-friendly for all types of users.
